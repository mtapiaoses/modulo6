Como mostrar un template con django.

Crear el archivo de la plantilla html: En la carpeta de tu proyecto Django, crea una nueva carpeta llamada "templates" si aún no existe. Dentro de la carpeta "templates", crea un archivo HTML para tu plantilla. Por ejemplo, podrías llamarlo "mi_template.html".
Configurar la ruta URL Abre el archivo "urls.py" de tu aplicación Django (generalmente ubicado en la carpeta del proyecto o en la carpeta de la aplicación). Importa la función "TemplateView" de Django: from django.views.generic import TemplateView.
Renderizar la plantilla Abre el archivo de vista asociado a la URL que configuraste en el paso anterior. Importa la función render de Django: from django.shortcuts import render. Define una función de vista que maneje la solicitud de esa URL
Mostrar la plantilla en el navegador Ejecuta tu servidor Django localmente (por ejemplo, mediante el comando python manage.py runserver). Abre tu navegador web y ve a la URL correspondiente a tu plantilla (por ejemplo, http://localhost:8000/mi_ruta/).
